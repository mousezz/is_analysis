#### 用例：  登录
- 权限：访客。
- 功能： 登录到平台。
- API请求地址： 接口基本地址/api/login
- 请求方式 ： post
- 请求实例：

```
  {   
      "id":"20150414221",
      "password":"48791ddc",
      "type":"学生"
  }

```
- 请求参数说明：

参数名称	| 说明
---|---
id|学生学号或者老师的工号，由type决定。
password|用户的密码，不是原文，是md5加密后的字符串
type|用户类型，学生或者老师

- 返回实例：
```
{         
      "status": true,
      "info": "success"
  }
```


参数名称 | 说明
---|---
status | bool类型，true表示正确的返回，false表示有错误
info | 返回结果说明信息
